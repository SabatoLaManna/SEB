
# Sabato La Manna - SEB tool - DODONA

Een programma dat ervoor zorgt dat je het internet kan gebruiken in de Safe Exam Browser




## Belangrijke Informatie
- Om dit te gebruiken moet je het `.seb` bestand dat gecreëerd wordt open doen in de plaats van het bestand dat je krijgt van de leerkracht. 
- Het bestand dat je krijgt start automatisch op in dodona.be, maar je kan de link overtypen naar bv. Smartschool bovenaan je scherm
- **Dit is niet gemaak om gebruikt te worden zonder toestemming van de leerkracht, en is enkel een proof of concept gemaakt voor eigen interesse en kennis**

## Installatie

1. Download python (lieftst versie 3.14) via de officiele [**python website**](https://www.python.org/downloads/) **TIJDENS DE INSTALLATIE ZAL GEVRAAGD WORDEN ALS JE PYTHON WILT TOEVOEGEN AAN __PATH__ KRUIS DIT VAKJE AAN**
2. Download deze repo als een `.zip` bestand door bovenaan op **code** te klikken, en daarna op **Download Zip**
3. Pak het `.zip` bestand uit, en ga de nieuwe, uitgepakte map in
4. Kijk zeker na dat Microsoft Edge in de correcte locatie staat, nl. ```C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe``` maar dat is standaard wel in orde. 
5. Dubbelklik op het bestand getiteld `setup.bat`
6. De installatie zal zichzelf nu uitvoeren, als deze compleet is kan je de console die geopend was sluiten. 
7. Navigeer naar je `\Downloads` map, als alles goed is gegaan is daar nu een bestand genaamd `Dodona_Seb_GPT.seb` dit is het bestand dat je moet gebruiken.
